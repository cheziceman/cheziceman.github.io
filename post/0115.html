2024-01-15 Dogma for SimpleWeb

  • Article 1 : SimpleWeb is not SlowWeb because someone used that as a trademark. But it is a web designed for everyone, everywhere, even with a slow network and a small bandwidth. It is more a friend of Permacomputing because you can use small or old terminals to do it.
  • Article 2 : SimpleWeb can be created directly in HTML or first any Markdown editor. For example, I used Zettlr or Typora but you can use Markor, Ghostwriter or even a Notepad. Then you can use any HTML converter. I recommend Pandoc but other tools are possible (for example Zettlr or Typora are using Pandoc tools)
  • Article 3 : Think SWING : SimpleWeb Is Not Gemini (or Gopher) … but it is as simple as those two protocols for the result.
  • Article 4 : SimpleWeb has no pictures. It is designed to promote love of writing and reading.
  • Article 5 : SimpleWeb use no SEO because SEO is dependant of GAFAM, and Search Engines.
  • Article 6 : SimpleWeb is independant of the Operating System. You can create your site on Windows, GNU/Linux, BSD, OSX, Android…Even MSDos if you find a way to convert markdown to HTML or if you write directly in HTML.
  • Article 7 : SimpleWeb don’t need CSS but you can only use some attributes as :
    • title
    • auto…
  • Article 8 : SimpleWeb don’t use social networks or comments. It is a social netowrk if you make linnks between other site and create answers as new articles in your own blog.
